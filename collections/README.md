# 脚本推荐计划

为了让用户更方便的找到一些有趣的脚本，我们会每隔一段时间出一次合集，整理和介绍这段时间出现的有趣的脚本。

当然这样能让用户看到一些有趣的内容，但这还不够，然而在 JSBox 内置脚本商店又容易碰到审核上的困难，所以我们也决定在之后要提供一个网页版的脚本目录，敬请期待。
